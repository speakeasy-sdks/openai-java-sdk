# FineTuneHyperparams

The hyperparameters used for the fine-tuning job. See the [Fine-tuning Guide](/docs/guides/fine-tuning/hyperparameters) for more details.


## Fields

| Field                                                                                                                                    | Type                                                                                                                                     | Required                                                                                                                                 | Description                                                                                                                              |
| ---------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------- |
| `batchSize`                                                                                                                              | *Long*                                                                                                                                   | :heavy_check_mark:                                                                                                                       | The batch size to use for training. The batch size is the number of<br/>training examples used to train a single forward and backward pass.<br/> |
| `classificationNClasses`                                                                                                                 | *Long*                                                                                                                                   | :heavy_minus_sign:                                                                                                                       | The number of classes to use for computing classification metrics.<br/>                                                                  |
| `classificationPositiveClass`                                                                                                            | *String*                                                                                                                                 | :heavy_minus_sign:                                                                                                                       | The positive class to use for computing classification metrics.<br/>                                                                     |
| `computeClassificationMetrics`                                                                                                           | *Boolean*                                                                                                                                | :heavy_minus_sign:                                                                                                                       | The classification metrics to compute using the validation dataset at the end of every epoch.<br/>                                       |
| `learningRateMultiplier`                                                                                                                 | *Double*                                                                                                                                 | :heavy_check_mark:                                                                                                                       | The learning rate multiplier to use for training.<br/>                                                                                   |
| `nEpochs`                                                                                                                                | *Long*                                                                                                                                   | :heavy_check_mark:                                                                                                                       | The number of epochs to train the model for. An epoch refers to one<br/>full cycle through the training dataset.<br/>                    |
| `promptLossWeight`                                                                                                                       | *Double*                                                                                                                                 | :heavy_check_mark:                                                                                                                       | The weight to use for loss on the prompt tokens.<br/>                                                                                    |
